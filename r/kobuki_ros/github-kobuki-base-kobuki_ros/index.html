README

Kobuki

This repository holds the ROS wrapper of Kobuki’s C++ driver plus various ROS tools and applications.

Documentation

  • Official Web Page - home page, sales, specifications and hardware howto.
  • Protocol, Usage and Api Documentation - in doxygen.
  • Ros Usage and Tutorials - on the roswiki.
  • Turtlebot Reference Platform - kobuki has been designed to meet the requirements of ROS REP #119 to support turtlebot.
