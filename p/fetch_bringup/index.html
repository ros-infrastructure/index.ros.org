  • Update launch files to support PS4 teleop w/ds4drv - Teleop: Change default /joy/dev param to /dev/fetch_joy to match udev rules

0.8.8 (2019-08-21)

  • Update launch files to support PS4 teleop w/ds4drv - Teleop: Change default /joy/dev param to /dev/fetch_joy to match udev rules

  • add joystick deadzone parameter (#42)
