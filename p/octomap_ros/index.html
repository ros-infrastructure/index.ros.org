octomap_ros

ROS package to provide conversion functions between ROS and OctoMap’s native types.

ROS1 support is on branch melodic-devel

ROS2 support is on branch ros2.

This repository continues from the groovy branch imported from SVN: http://alufr-ros-pkg.googlecode.com/svn/branches/octomap_stacks-groovy-devel/octomap_ros/ See https://code.google.com/p/alufr-ros-pkg/ for the previous versions.

0.4.1 (2020-04-23)

  • Added archive destination for static libraries (#10)
  • Fix catkin_lint issues and cmake policy CMP0038 (#9)
  • Update maintainer emails
  • Contributors: Andrea Ponza, Armin Hornung, Sebastian Kasperski, Wolfgang Merkt

0.4.0 (2014-04-17)

  • Dropped PCL support in favor of sensor_msgs::PointCloud2.

  • Removed deprecated OctomapROS class
