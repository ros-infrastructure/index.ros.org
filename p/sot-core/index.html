README.md

sot-core

This software provides a set of dynamic graph plug-ins which can be used to define and solve hierarchical tasks.

Setup

To compile this package, it is recommended to create a separate build directory:

mkdir _build
cd _build
cmake [OPTIONS] ..
make install

Please note that CMake produces a CMakeCache.txt file which should be deleted to reconfigure a package from scratch.

README.md

sot-core

This software provides a set of dynamic graph plug-ins which can be used to define and solve hierarchical tasks.

Setup

To compile this package, it is recommended to create a separate build directory:

mkdir _build
cd _build
cmake [OPTIONS] ..
make install

Please note that CMake produces a CMakeCache.txt file which should be deleted to reconfigure a package from scratch.
