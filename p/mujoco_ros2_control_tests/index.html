CHANGELOG

Changelog for package mujoco_ros2_control_tests

0.0.3 (2026-05-01)

  • Switch to forward_command_controller dependency (#173)
  • Add epsilon to avoid floating point comparison (#166)
  • Ensure the clock messages are flushed before time checking (#153)
  • [Feature] Add ability to run the simulation progress by steps upon pause (#139)
  • Contributors: Christoph Fröhlich, Erik Holum, Sai Kishor Kothakota

0.0.2 (2026-03-17)

  • Fix reset simulation service for robots using PID control (#140)
  • Use pixi-build-ros as the backend for pixi builds (#130)
  • Improve integration test reliability and speed for CI (#131)
  • Rename default node to mujoco_ros2_control_node (#132)
  • Contributors: Erik Holum, Sai Kishor Kothakota

0.0.1 (2026-02-24)

  • Updates docs and comments (#113)
  • Add mujoco_ros2_control_demos package and retarget tests (#100)
  • Containerize pixi in CI (#99)
  • Adjust pixi CI job and bump versions (#96)
  • Add ResetWorld service to reset to a specific defined keyframe (#95)
  • Re-enable rolling workflow (#91)
  • Separate tests to mujoco_ros2_control_tests package (#90)
  • Contributors: Erik Holum, Louis LE LAY, Sai Kishor Kothakota
