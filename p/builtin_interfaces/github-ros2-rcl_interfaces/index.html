builtin_interfaces/README.md

builtin_interfaces

A package containing message definitions for types defined in the OMG IDL Platform Specific Model.

For more information about ROS 2 time, see design.ros2.org.

For more information about ROS 2 interfaces, see docs.ros.org.

Messages (.msg)

  • Duration: Describes a time duration composed of seconds and nanoseconds components.
  • Time: Describes a point in time, composed of seconds and nanoseconds components.

Quality Declaration

This package claims to be in the Quality Level 1 category, see the Quality Declaration for more details.

builtin_interfaces

A package containing message definitions for types defined in the OMG IDL Platform Specific Model.

For more information about ROS 2 time, see design.ros2.org.

For more information about ROS 2 interfaces, see docs.ros.org.

Messages (.msg)

  • Duration: Describes a time duration composed of seconds and nanoseconds components.
  • Time: Describes a point in time, composed of seconds and nanoseconds components.

Quality Declaration

This package claims to be in the Quality Level 1 category, see the Quality Declaration for more details.
