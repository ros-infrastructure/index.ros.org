unique_identifier_msgs

ROS 2 messages for Universally Unique Identifiers.

For more information about UUIDs, see wikipedia and the UUID specification.

For more information about ROS 2 interfaces, see interfaces documentation.

Messages (.msg)

  • UUID: A universally unique identifier (UUID).

Quality Declaration

This package claims to be in the Quality Level 1 category, see the Quality Declaration for more details.
